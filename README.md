# Coca-Cola-Project

📖 Project Overview

This project focuses on analyzing Coca-Cola sales data across multiple retailers and regions using Microsoft Excel. The main objective is to extract meaningful insights from sales data and create an interactive dashboard that highlights key performance indicators (KPIs), trends, and profitability metrics.

The analysis helps identify top-performing retailers, regional performance differences, and overall sales growth patterns to support data-driven business decisions.

🎯 Project Objectives
Analyze total sales across retailers and regions
Identify top-performing retailers
Calculate total operating profit
Track sales trends over time
Compare regional performance
Build an interactive Excel dashboard
Create KPI cards for quick business insights
📂 Dataset Description

The dataset contains Coca-Cola sales transaction data from multiple retailers.

Key Columns Used:
Retailer — Name of retailer selling products
Region — Sales region
Product — Product category
Sales — Total sales amount
Units Sold — Quantity sold
Operating Profit — Profit earned
Date — Sales transaction date

🧹 Data Cleaning Steps

The following steps were performed to clean the data:

Removed duplicate records
Handled missing values
Standardized retailer names
Converted date format
Verified numerical fields
Created calculated columns where required
📊 Data Analysis Process

The data analysis involved:

Grouping data by retailer
Calculating total operating profit
Summarizing regional sales
Creating pivot tables for aggregation
Identifying top retailers
Generating monthly sales trends

Retailer profits were aggregated into single rows using SUMIF/Pivot Tables to create summary-level insights.

📈 Dashboard Features

The Excel dashboard includes:

KPI Cards:

Profit by Retailer

Top Performing Retailers
Interactive Filters (Slicers)
🔍 Key Insights

Some key insights derived from the analysis:

Certain retailers contribute significantly to total profit
Regional sales performance varies across locations
Monthly trends highlight peak sales periods
Top retailers drive majority of revenue
📌 Business Recommendations

Based on the analysis:

Focus on high-performing retailers
Improve strategies in low-performing regions
Optimize inventory for peak months
Increase marketing in high-profit areas
